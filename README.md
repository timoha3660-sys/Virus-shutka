# Virus-shutka
Не настоящий вирус

Когда вы включите программу будет короткий звуковой сигнал и будет предупреждение:

ЭТО НЕ ВИРУС! ЭТО ШУТКА!
Если вы нажмёте "Да" - то
тогда начнётся хаос! Что-
бы остановить это - надо
выключить компьютер через
кнопку выключения на кор-
пусе компьютера.



Удачи!!

если вы нажмете "Нет" - то тогда программа закроется, а если нажмёте "Да" - то будут:
+Появлятся Крестики и треугольники на экране
+Через 60сек будет не настоящий BSOD
+Через 10сек полсле старта BSOD Компьютер перезагрузится.
